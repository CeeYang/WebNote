HTML 表格：
	table       : 标签定义 html 表格，
	border      : 边宽；默认为0；
	caption     : 定义表格的标题
	tr          : 定义表格的行
	td          : 定义表格单元，左对齐显示；
	th          : 定义表格的表头，居中显示；
	cellpadding : 表格内边距；
	cellspacing : 表格外边距，默认大概2px;
	colspan     : 合并单元格；

标题显示方式：

水平标题
Name: Age: Phone:
CeeYang 23 18888888888

垂直标题
Name: CeeYang
Age: 23
Phone: 18888888888

对齐方式:

使用 th 居中显示
Name: CeeYang
Age: 23
Phone: 18888888888

使用 td 左对齐显示
Name: CeeYang
Age: 23
Phone: 18888888888
